Recopilatorio Musical

Recopilatorio musical es una base de datos en la que se puede encontrar a artistas, grupo o albunes.

Tech

Recopilatorio musical utliza las siguientes tecnologías:

  • Node JS
  • MongoDB
  • Express JS

Installation

  1. Clonar el repositorio
    github.com/ylearven/proyecto7.git
  2. Instala las dependecias
    nodemon - express - mongoose - cors - dotenv

EndPoints

Usuarios

  1. Obtener todos
    • GET /api/v1/users
  2. Iniciar sesión
    • POST /api/v1/users/login
  3. Registrarse
    • POST /api/v1/users/register
  4. Actualizar rol
    • PUT /api/v1/users/:id
  5. Actualizar usuario
    • PUT /api/v1/users/:id
  6. Eliminar usuario
    • DELETE /api/v1/users/:id

Album

  1. Obtener todos los albunes
    • GET /api/v1/albumRouter
  2. Obtener por ID
    • GET /api/v1/albumRouter/:id
  3. Obtener por Nombre
    • GET /api/v1/albumRouter/Nombre/:Nombre
  4. Obtener por año de lanzamiento
    • GET /api/v1/albumRouter/Lanzamiento/:Lanzamiento
  5. Publicar un album
    • POST /api/v1/albumRouter
  6. Actualizar un album
    • PUT /api/v1/albumRouter/:id
  7. Borrar un album
    • DELETE /api/v1/albumRouter/:id

Modelo - Album

  • Nombre: String - Nombre del Album
  • Imagen:String - URL la portada del album
  • Lanzamiento: Number Año de lanzamiento del Album

Modelo - Genero

  • Nombre: String - Nombre del Genero
  • Grupo:Array(ObjectID) - Grupos que pertenecen a ese género
  • Solistas: Array(ObjectID) Solistas que pertenecen a este género

Modelo - Grupo

  • Nombre: String - Nombre del Grupo
  • Imagen:String - URL Imagen del grupo
  • Componentes:String - Integrantes del grupo
  • Debut: Number Año en el que debutaron como grupo
  • Album: Array(ObjectID) Último album del grupo

Modelo - Solista

  • Nombre: String - Nombre del Solista
  • Imagen:String - URL Imagen del solista
  • Debut: Number Año en el que debutó como solista
  • Album: Array(ObjectID) Último album
